# Stock Trading Using Vue :: Backend


## Feature

``` bash
  1. Display stock in graph
  2. Buy Stock (WIP)
  3. Sell Stock (WIP)
  4. Open data for MACHINE LEARNING (WIP)
```

## Prerequisite

``` bash
  # Recommend
  python: 3.6.*  
  django: 2.0.*
  djangorestframework 3.7.*  
  django-cors-headers 2.1.*  
  bcrypt 3.1.*
```

## Build Setup

``` bash
# run in local
  python manage.py runserver [port: optional]
```
